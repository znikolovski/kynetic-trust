## Design Context

This project uses the **Kinetic Trust / Neon Glass** design system (register: brand).

- `PRODUCT.md` — strategic context: register, users, positioning, anti-references, design principles
- `DESIGN.md` — visual system spec: color tokens, typography, elevation, components, Do's and Don'ts
- `.impeccable/design.json` — sidecar: rendered component snippets, tonal ramps, shadow/motion tokens, full narrative

Read all three before generating new screens, sections, or components. The frontmatter YAML in `DESIGN.md` is normative for token values; prose contextualizes how to apply them. The North Star is **"The Neon Vault"** — every visual decision should be legible from that metaphor.
